# Android Studio for Arch Linux ARM aarch64

This is a AUR package for Arch Linux ARM aarch64 packaging the [official releases](https://developer.android.com/studio#downloads) as convience and integrating with any AUR compatible package manager. Find all the details [aur.archlinux.org/packages/android-studio-aarch64/](https://aur.archlinux.org/packages/android-studio-aarch64/).

The [Github Repo](https://github.com/VeryBaaad/arch-aur-android-studio-aarch64) is [a mirror of the official repo](https://aur.archlinux.org/cgit/aur.git/log/?h=android-studio-aarch64), but enables anyone to send pull requests.

Fork from [Github Repo](https://github.com/kordianbruck/arch-aur-android-studio).
